Enterprise Build (Most Secure)

PyInstaller binary compilation for maximum enterprise security

1. Build with PyInstaller
2. Configure Azure credentials
3. Deploy in enterprise environment
docker build -t aks-cost-optimizer . && docker run -d -p 5000:5000 aks-cost-optimizer

Standard Docker

Containerized deployment for development and testing environments

1. Pull the container image
2. Set environment variables
3. Access enterprise dashboard
docker run -d -p 5000:5000 -e AZURE_TENANT_ID=xxx aks-cost-optimizer

Air-Gapped Deployment

Secure enterprise deployment with no external dependencies

1. Download enterprise package
2. Configure offline environment
3. Deploy with enterprise settings
./deploy-enterprise.sh --mode=airgapped --license=enterprise
